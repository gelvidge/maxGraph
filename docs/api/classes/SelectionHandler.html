Class SelectionHandler

Graph event handler that handles selection. Individual cells are handled separately using or one of the edge handlers. These handlers are created using <mxGraph.createHandler> in <mxGraphSelectionModel.cellAdded>.

To avoid the container to scroll a moved cell into view, set to false.

Constructor: mxGraphHandler

Constructs an event handler that creates handles for the selection cells.
