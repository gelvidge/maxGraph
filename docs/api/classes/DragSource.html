mouseDown

  • mouseDown(evt: MouseEvent): void
    • Defined in view/other/DragSource.ts:300

    Returns the drop target for the given graph and coordinates. This implementation uses {@link mxGraph.getCellAt}.

    To ignore popup menu events for a drag source, this function can be overridden as follows.

    example
    var mouseDown = dragSource.mouseDown;

    dragSource.mouseDown(evt)
    {
      if (!mxEvent.isPopupTrigger(evt))
      {
        mouseDown.apply(this, arguments);
      }
    };

    Parameters

    • evt: MouseEvent

    Returns void
