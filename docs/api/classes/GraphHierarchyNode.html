    Visits all cells recursively and applies the specified filter function to each cell. If the function returns true then the cell is added to the resulting array. The parent and result paramters are optional. If parent is not specified then the recursion starts at {@link root}.

    Example: The following example extracts all vertices from a given model:

    var filter(cell)
    {
        return model.isVertex(cell);
    }
    var vertices = model.filterDescendants(filter);

    Parameters

      JavaScript function that takes an Cell as an argument and returns a boolean.
