Class CellMarker

A helper class to process mouse locations and highlight cells.

Helper class to highlight cells. To add a cell marker to an existing graph for highlighting all cells, the following code is used:

let marker = new mxCellMarker(graph);
graph.addMouseListener({
  mouseDown: ()=> {},
  mouseMove: (sender, me)=>
  {
    marker.process(me);
  },
  mouseUp: ()=> {}
});

Event: mxEvent.MARK

Fires after a cell has been marked or unmarked. The state property contains the marked or null if no state is marked.

Portion of the width and hight where a state intersects a given coordinate pair. A value of 0 means always highlight. Default is <mxConstants.DEFAULT_HOTSPOT>.
