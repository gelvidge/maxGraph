Static RectanglePerimeter

  • RectanglePerimeter(bounds: Rectangle, vertex: CellState, next: Point, orthogonal?: boolean): Point
    • Defined in view/style/Perimeter.ts:90

    Describes a rectangular perimeter for the given bounds.

    Parameters

    • bounds: Rectangle

      {@link mxRectangle} that represents the absolute bounds of the vertex.

    • vertex: CellState

      CellState that represents the vertex.

    • next: Point

      {@link mxPoint} that represents the nearest neighbour point on the given edge.

    • orthogonal: boolean = false

      Boolean that specifies if the orthogonal projection onto the perimeter should be returned. If this is false then the intersection of the perimeter and the line between the next and the center point is returned.

    Returns Point
